# acelynn qiao · personal site

personal portfolio & about page. hosted on github pages at [aceyq.github.io](https://aceyq.github.io).

built with vanilla html, css, and js — no frameworks, no build tools, just vibes.

## structure

```
acelynn-site/
├── index.html        # home / landing
├── about.html        # about me
├── style.css         # all styles
├── script.js         # animations (fireflies, dog, rotating text)
└── Acelynn_Qiao_Resume.pdf   # resume (add this yourself)
```

## local dev

just open `index.html` in your browser — or use the live server extension in vs code for hot reload.

## to update

1. edit the files
2. `git add .`
3. `git commit -m "your message"`
4. `git push`

github pages auto-deploys from the `main` branch. changes go live in ~30 seconds.
